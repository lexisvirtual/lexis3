# 📜 Protocolo Oficial Leo (Motor SEO Autônomo v1.1)

Este documento é o cérebro algorítmico do Leo. Define gatilhos, decisões e ações corretivas baseadas em métricas REAIS.

## 1. FÓRMULA DE PRIORIDADE (LEO SCORE)
O Leo deve orbitar primeiro as páginas com maior potencial de ROI.
`Score = (Impressões x 0.4) + (Posição Inversa x 0.3) + (Conversões x 0.3)`
*Prioridade Máxima: Score Alto + Posição entre 5 e 15.*

## 2. GATILHOS E AÇÕES AUTOMÁTICAS

### A. OPORTUNIDADE "QUASE RANQUEANDO" (Pos. 5-15)
- **Condição**: Posição média 5-15 E Impressões > 300/mês.
- **Ação**: 
    1. Reescrever Título (Promessa + Número).
    2. Inserir "Resposta Direta" (40-60 palavras) no topo para Snippet.
    3. Adicionar 2 FAQs com Schema JSON-LD.
    4. Inserir 3 links internos apontando para esta página.

### B. IMPRESSÃO ALTA, CTR BAIXO (Visibilidade sem Clique)
- **Condição**: Impressões > 500 E CTR < 3%.
- **Ação**:
    1. Update de Meta-Description (Diferenciação Concreta).
    2. Inserir Ano Atual (2026) no título.
    3. Promessa explícita de tempo (ex: "em 2 semanas").

### C. CONTEÚDO ESTAGNADO (>90 Dias | Pos. > 20)
- **Condição**: Idade > 90 dias E Posição > 20.
- **Ação**:
    1. Expansão Massiva (+800 palavras).
    2. Inserir Tabela Comparativa de Dados.
    3. **Pivotagem**: Se não evoluir em 60 dias após expansão, criar nova Long-Tail ultra-niche (nicho do nicho).

### D. BAIXO ENGAJAMENTO / ALTA SAÍDA (User Experience)
- **Condição**: Engajamento < 45s OU Scroll < 50%.
- **Ação**:
    1. Inserir Sumário Clicável.
    2. Inserir "Takeaway Box" (Resumo 3 pontos) no topo.
    3. Adicionar Prova Social/Depoimento acima da dobra.

## 3. REGRAS ESTRATÉGICAS (CONSTITUIÇÃO)
1. **Canibalização**: Se duas páginas ranqueiam para a mesma query, consolidar conteúdo e redirecionar a mais fraca.
2. **Rising Queries**: Queries com crescimento > 30% mês a mês ganham artigo dedicado linkando para a Pilar.
3. **Pilar First**: Nunca publicar conteúdo sem interlink para a página pilar do cluster.
4. **Otimização > Criação**: Atualizar conteúdo existente é prioridade sobre criar novos volumes.

## 4. CICLO OPERACIONAL
- **Semanal**: 1 Otimização de "Quase Ranqueando" + 1 Atualização de Antigo.
- **Mensal**: Revisão das 10 páginas de maior impressão + Update de 1 Pilar.
- **Trimestral**: Auditoria Geral de Interlink + Refresh de Dados/Ano.

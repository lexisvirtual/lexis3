## Informações do Projeto

**Nome:** Metodologia Lexis - Inglês como Habilidade

**Descrição:** Landing page de alta conversão para o Método Lexis, focada em transformar o aprendizado de inglês em um esporte cognitivo de alto desempenho.

**Tecnologias:**
- React 18
- TypeScript 5.8.2
- Vite 6.2.0
- Tailwind CSS

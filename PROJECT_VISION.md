# PROJECT VISION: LEXIS V9.1 - THE AUTONOMOUS PERFORMANCE ENGINE

## 1. LINHA LÓGICA COMPLETA (THE STRATEGIC LINE)
A estratégia Lexis é um organismo vivo composto por três camadas interdependentes que garantem a aquisição de clientes, a entrega de resultados e a escala operacional:

**CORE PEDAGÓGICO (O PRODUTO)**
> "Idioma não se aprende. Idioma se treina."
O inglês é tratado como uma **habilidade motora** (memória procedural), não como conteúdo acadêmico. A Imersão de 120h é o mecanismo único para quebrar a inércia e criar caminhos neurais de fala.

**MECANISMO DE MARKETING (A AQUISIÇÃO)**
> "A Narrativa de Inocência do Aluno."
Atribuímos o fracasso anterior do aluno aos métodos tradicionais (gramática excessiva/lentidão) e posicionamos a imersão como a única solução lógica. O SEO Programático escala essa narrativa capturando leads em momentos de dor.

**MOTOR TÉCNICO (A ESCALA)**
> "Protocolos Leo e Roger."
Uma infraestrutura autônoma que utiliza agentes de IA para gerar, auditar e otimizar ativos digitais (posts, imagens, SEO) sem intervenção humana, garantindo autoridade topical perpétua.

---

## 2. ETAPAS DO RACIOCÍNIO (THE REASONING)

1.  **Identificação do Gap:** O mercado de ensino de inglês falha porque tenta ensinar uma habilidade motora (falar) através de conhecimento declarativo (regras gramaticais). Isso gera o "estudante eterno" que entende, mas não fala.
2.  **O Mecanismo Único:** Propomos a Imersão de 120h (2 semanas, 10h/dia). Essa densidade extrema simula a vivência no exterior e força o cérebro a migrar do modo "tradução" para o modo "produção automática".
3.  **Escala via SEO Programático:** Em vez de depender de anúncios caros, ocupamos as lacunas do Google (SERPs fracas) com conteúdo de alta qualidade gerado por IA que já pré-educa o lead na metodologia Lexis.
4.  **Governança de Qualidade Autônoma:** Através do agente Roger (Auditor) e do protocolo Leo (SEO), garantimos que cada post publicado mantenha o padrão "Elite" (IPL > 90), corrigindo-se e reciclando-se automaticamente.

## 4. DESENVOLVIMENTO DOS TÓPICOS

### 4.1 CORE PEDAGÓGICO: INGLÊS COMO HABILIDADE MOTORA
A Lexis ignora a progressão linear (A1->C2) das escolas tradicionais. Focamos na **Inteligibilidade e Velocidade de Resposta**.
- **Start (Fonética):** Base para ser entendido.
- **Run (Estruturas):** Os "motores" da frase.
- **Fly (Vocabulário):** 600-700 palavras de alta frequência.
- **Liberty (Conversação):** Simulações de alta pressão.

### 4.2 MECANISMO DE MARKETING: NARRATIVA DE INOCÊNCIA E SEO
O marketing não vende "aulas", vende a "saída da prisão do método tradicional".
- **SEO Programático:** Não criamos blogs, alocamos páginas. Cada post é uma porta de entrada para o funil que valida a dor do aluno e oferece a Imersão como cura.
- **Autoridade Topical:** Ao dominar clusters inteiros (ex: Inglês para Reuniões, Inglês para TI), o Google nos reconhece como a fonte definitiva.

### 4.3 MOTOR TÉCNICO: AGENTES LEO E ROGER
- **Leo (O Estrategista SEO):** Monitora o Search Console e a planilha de pautas. Decide quais posts precisam de upgrade, FAQ ou interlink baseado na fórmula: `Score = (Impressões x 0.4) + (Posição Inversa x 0.3) + (Conversões x 0.3)`.
- **Roger (O Guardião da Qualidade):** Executa o audit pedagógico. Se o post não atinge o **IPL (Indicador de Performance Linguística)** necessário, ele ordena a reescrita ou exclusão.
- **Infraestrutura V9.1:** Cloudflare Workers orquestrando o GitHub. Imagens são processadas, convertidas para WebP e salvas localmente para garantir 100% de disponibilidade e performance (Core Web Vitals).

### 4.4 O SISTEMA DE EXECUÇÃO (PUBLISHER V8.0+)
O pipeline de publicação é 100% automatizado:
1.  **Triage:** Escolha do tema baseado no cluster.
2.  **Writing:** Redação seguindo o `PADRAO_BLOG_ROGER_LEO.md`.
3.  **Auditing:** Roger valida pedagogia e tom de voz.
4.  **Imaging:** Busca Pixabay -> Otimização WebP -> Commit Local.
5.  **Publishing:** Commit no GitHub -> Build Vite -> Deploy GitHub Pages.

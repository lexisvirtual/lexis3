# PROJECT VISION: LEXIS ACADEMY V9.1

## 1. LINHA LÓGICA (O CORE ESTRATÉGICO)
A Lexis Academy opera em uma tríade simbiótica onde cada pilar sustenta o próximo:

**PEDAGOGIA (O Coração)** → **MARKETING (O Mecanismo)** → **TÉCNICO (O Motor)**

1.  **Pedagogia:** O inglês é tratado como **Habilidade Motora Procedural**, não como conteúdo acadêmico. O objetivo é a automação da fala através de alta densidade (120h de imersão).
2.  **Marketing:** Utiliza a **Narrativa de Inocência do Aluno**. Se o aluno não fala, a culpa é do método tradicional (gramatical/lento). A Lexis é apresentada como o "Mecanismo Único" de correção.
3.  **Técnico:** Uma infraestrutura autônoma (Leo & Roger Agents) que gera autoridade topical via **SEO Programático**, educando e convertendo leads através de workshops de "Elite".

---

## 2. ETAPAS DO RACIOCÍNIO (A JORNADA DA SOLUÇÃO)

1.  **Diagnóstico do Erro Sistêmico:** Identificamos que o ensino tradicional foca na memória declarativa (saber sobre a regra), enquanto a fluência exige memória procedural (saber fazer/falar).
2.  **Mecanismo de Ruptura (Imersão):** A solução é a compressão de tempo. 120 horas em 2 semanas forçam o cérebro a abandonar a tradução mental, algo que 2 horas semanais em 5 anos não conseguem.
3.  **Escalabilidade de Autoridade:** Para atrair o público certo, criamos uma rede de conteúdo que não "vende curso", mas "treina o aluno" antecipadamente. Cada post é um micro-treinamento de IPL (Indicador de Performance Linguística).
4.  **Governança de Qualidade:** O conteúdo não pode ser genérico. O **Roger (Auditor)** garante que o DNA Lexis (tensão executiva, bilinguismo funcional e ancoragem lexical) esteja presente em 100% das páginas.

## 4. ESTRATÉGIAS DE FORTALECIMENTO (MOATS)

-   **EMA (Exponential Moving Average) Consciousness:** O sistema monitora a qualidade média dos posts. Se a média sobe, o "Threshold de Aprovação" do Roger sobe automaticamente, forçando o Leo a escrever melhor.
-   **Ancoragem Lexical Reversa:** Termos ensinados na seção de vocabulário (Nível 1) SÃO OBRIGATORIAMENTE usados nos cenários de prática (Níveis 2 e 3) e destacados em negrito. Isso prova a funcionalidade do método.
-   **Self-Hosting de Ativos:** Imagens são processadas via `wsrv.nl`, convertidas para WebP e hospedadas localmente no repositório para garantir 100% de disponibilidade e velocidade (LCP).
-   **Narrativa de Inocência:** Todo post deve reforçar que o aluno é capaz, mas foi "vítima" de um sistema de ensino ineficiente.

## 5. DESENVOLVIMENTO DOS TÓPICOS

### 5.1 O Core Pedagógico: Habilidade Motora
A metodologia Lexis baseia-se na neurociência da aquisição de linguagem. O "Idioma não se aprende, se treina" significa que a sala de aula é um ginásio, não uma biblioteca.
-   **Foco:** Resposta automática, redução do delay de tradução.
-   **Métrica:** IPL (Indicador de Performance Linguística).

### 5.2 O Mecanismo de Marketing: SEO Programático
Em vez de buscar "palavras-chave", buscamos "intenções de busca". Alocamos páginas para resolver dores específicas do executivo moderno (Ex: "Como lidar com interrupções em calls").
-   **Leo Strategy:** Prioriza URLs com alto volume de impressões, mas posição estagnada, aplicando "Upgrades de Elite".

### 5.3 O Motor Técnico: Agentes Autônomos (V9.1)
O **Lexis Publisher Worker** não é um simples script. É um ecossistema:
-   **Leo (SEO):** O estrategista que escolhe os temas e redige os workshops.
-   **Roger (Audit):** O guardião que rejeita conteúdos que não atingem a nota mínima de IPL (atualmente 85/100 para Elite).
-   **Ana (Scheduler):** Orquestra o timing das publicações para manter o "frescor" do site perante os motores de busca.

# 🎯 PROJECT VISION: LEXIS ACADEMY V9.1

Este documento é a "Fonte Única da Verdade" (Single Source of Truth) para a estratégia Lexis. Ele consolida a lógica pedagógica, o mecanismo de marketing e a infraestrutura técnica em um ecossistema coeso.

---

## 1. 🏗️ LINHA LÓGICA COMPLETA
O projeto opera em um fluxo de dependência onde cada camada fortalece a próxima:

1.  **CORE PEDAGÓGICO (O "O QUE"):** Inglês como Habilidade Motora. O foco não é aprender conteúdo, mas treinar o cérebro para resposta automática.
2.  **MECANISMO DE MARKETING (O "COMO VENDE"):** Narrativa de Inocência do Aluno + SEO Programático. Ataca o método tradicional para posicionar a imersão como a única solução lógica.
3.  **MOTOR TÉCNICO (O "COMO ESCALA"):** Protocolo Leo (SEO) + Roger Agent (Audit). Automação de ativos digitais que geram autoridade topical com custo marginal zero.

---

## 2. 🧠 ETAPAS DO RACIOCÍNIO (THE STRATEGIC CHAIN)

1.  **Identificação do Gap:** O mercado tradicional ensina inglês como matéria acadêmica (conhecimento declarativo). Alunos "sabem" a regra, mas não "falam" sob pressão.
2.  **A Proposta de Valor:** Transformar o inglês em habilidade procedural (como dirigir ou tocar piano). Isso exige alta intensidade (120h em 2 semanas) e ambiente de imersão.
3.  **A Engenharia de Aquisição:** Em vez de depender apenas de tráfego pago, o sistema aloca páginas de alta autoridade onde o Google apresenta resultados fracos, usando SEO programático.
4.  **A Automação de Governança:** O uso de IA não é para "escrever blogs", mas para gerenciar um ecossistema de conteúdo que se auto-audita, se auto-corrige e se auto-promove.

---

---

## 4. 🛡️ ESTRATÉGIAS DE FORTALECIMENTO (MITIGAÇÃO)

*   **Roger Agent (Content Guardian):** Auditoria obrigatória de cada post com métricas de IPL (Indicador de Performance Linguística). Se o post não atingir o nível de elite, ele é reciclado ou deletado.
*   **Protocolo Leo (ROI-Driven SEO):** Priorização de edição e melhoria de posts baseada em dados reais de conversão e posição inversa, não apenas volume de busca.
*   **Self-Hosted Assets:** Armazenamento local de imagens otimizadas (WebP) para garantir longevidade do conteúdo e velocidade de carregamento (Core Web Vitals).
*   **Git Rebase Automation:** Scripts que resolvem conflitos de commit automaticamente, garantindo que a máquina de publicação nunca pare.

---

## 5. 📚 DESENVOLVIMENTO DOS TÓPICOS

### 5.1 Pedagogy: The Motor Skill Framework
A Lexis ignora a gramática expositiva. O treino é dividido em 4 fases: **Start** (Fonética), **Run** (Estruturas), **Fly** (Vocabulário) e **Liberty** (Conversação). O objetivo é a redução da tradução mental através da repetição deliberada.

### 5.2 Marketing: Programmatic SEO & Innocence Narrative
O marketing utiliza o SEO para "educar" o lead. Cada post segue o padrão de 9 seções, começando sempre inocentando o aluno ("O erro não é seu, é do método"). O sistema ataca queries de "intenção de dor" para converter leitores em prospectos.

### 5.3 Technical: The Leo/Roger Engine
O sistema é composto por Cloudflare Workers que atuam como agentes autônomos:
*   **Leo:** Analisa o Search Console e decide quais páginas precisam de upgrade.
*   **Roger:** Garante que o conteúdo siga o padrão C1+ de vocabulário e a estrutura pedagógica de 3 níveis (Mapping -> Cenário -> Missão).

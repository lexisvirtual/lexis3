# PROJECT VISION: Lexis V9.1 - O Sistema de Aquisição Orgânica Autônomo

## 1. Linha Lógica (The Golden Thread)
A Lexis não é uma escola de inglês, é uma empresa de tecnologia e marketing movida por uma pedagogia disruptiva. A lógica segue este fio condutor:

*   **Pedagogia**: O inglês é uma habilidade motora (procedural), não um conhecimento acadêmico (declarativo). "Idioma não se aprende, se treina".
*   **Marketing**: "Narrativa de Inocência". O fracasso do aluno no passado não é culpa dele, mas de um método falho (tradicional). A Lexis oferece o "Mecanismo Único" (Imersão 120h) para resolver isso.
*   **Técnico**: Automação total via Agentes de IA (Leo & Roger). Escala massiva de conteúdo SEO que educa o lead e valida a metodologia antes mesmo do primeiro contato comercial.

---

## 2. Etapas do Raciocínio (Strategic Stages)

### I. Identificação do Gap
O mercado está saturado de cursos que ensinam gramática mas não geram fala. O gap está na transição entre o *entender* e o *produzir* sob pressão.

### II. O Mecanismo Único (Unique Mechanism)
A Imersão de 120 horas em 2 semanas. É o choque necessário para forçar o cérebro a migrar o processamento da linguagem para a memória procedural.

### III. Escala de Aquisição (Scale)
Uso de Programmatic SEO para dominar clusters de intenção (Business English, Fluência, Medo de Falar). O sistema aloca páginas onde o Google é fraco, reduzindo o CAC (Custo de Aquisição de Cliente) ao mínimo.

### IV. Governança Autônoma (Governance)
Agentes IA (Leo e Roger) garantem que cada post publicado mantenha o padrão de elite, sem intervenção humana, criando um ativo digital que cresce organicamente.

## 4. Desenvolvimento dos Tópicos

### 4.1 Pedagogia: Motor Skill Theory
A base científica da Lexis. Focamos na "Prática Deliberada" e "Repetição Espaçada". O aluno é treinado como um atleta. O conteúdo gerado pelos agentes deve sempre refletir isso, tratando o erro como uma ferramenta de ajuste e não como falha.

### 4.2 Marketing: A Narrativa de Inocência
Fundamental para quebrar a barreira de entrada. Ao dizer que o aluno é inocente, removemos o peso emocional do fracasso anterior. O blog serve como uma jornada de doutrinação: ele entra buscando uma dica de gramática e sai convencido de que precisa de uma imersão de 120h.

### 4.3 Técnico: O Ecossistema Leo & Roger
*   **Leo (SEO Strategy)**: O estrategista. Ele olha para os dados (Google Sheets), decide o que é prioridade e escreve o conteúdo usando prompts de elite (Apple Style).
*   **Roger (Quality Guardian)**: O auditor. Ele não deixa passar nada que não seja C1/C2 ou que falte bilinguismo funcional. Ele comanda o Leo para reescrever até atingir a nota de corte.
*   **Infrastructure**: Cloudflare Workers + GitHub Actions + Vite SPA. Uma stack serverless, extremamente rápida e de custo quase zero de manutenção.

O Lexis V9.1 é a materialização de uma máquina de crescimento infinita. Onde o conteúdo educa, o SEO atrai e a tecnologia escala o que antes dependia de humanos.

# 🎯 PROJECT VISION: LEXIS ACADEMY V9.1

## 1. Linha Lógica (The Golden Thread)
**Pedagogia (Motor Skill)** → **Marketing (Narrativa de Inocência)** → **Técnico (Leo/Roger Agents)**

A Lexis Academy não é uma escola de idiomas; é um centro de treinamento de alta performance. A lógica flui da premissa pedagógica de que o inglês é uma habilidade motora, passa por um marketing que remove a culpa do aluno, e é escalada por uma infraestrutura técnica autônoma que garante qualidade e SEO.

---

## 2. Etapas do Raciocínio

### I. O Gap Habilidade vs. Conhecimento
Identificamos que o mercado tradicional ensina inglês como *conhecimento declarativo* (regras, fatos), enquanto a fluência exige *conhecimento procedural* (habilidade motora/automática).

### II. O Mecanismo Único: Imersão 120h
A solução para converter conhecimento em habilidade é a densidade de treino. 120 horas em 2 semanas criam a pressão necessária para o cérebro automatizar processos que levariam anos no modelo tradicional.

### III. A Escala via Programmatic SEO
Para alimentar esse mecanismo, usamos clusters de conteúdo que atacam dores específicas do executivo e do estudante, utilizando o Leo Protocol para dominar o tráfego orgânico.

### IV. A Governança de IA (Roger Agent)
A escala não pode sacrificar a qualidade. O Roger Agent audita cada peça para garantir que o tom de voz, a precisão cultural e a "Narrativa de Inocência" sejam preservados.

## 4. Estratégias de Fortalecimento

1. **Self-Hosting de Imagens:** Migração para armazenamento local em `public/img/posts/` via `wsrv.nl` para eliminar links quebrados.
2. **Estrutura de 9 Seções:** Obrigatoriedade de tabelas, planos de treino e IPL Self-Check em todos os posts para evitar conteúdo genérico.
3. **Roger Audit v3.5:** Auditoria rigorosa de bilinguismo e "Narrativa de Inocência" antes do commit.
4. **Git Rebase Automation:** Scripts que realizam `pull --rebase` antes de qualquer push autônomo para garantir integridade do repositório.
5. **Fórmula LEO Score:** Priorização de posts com alto ROI baseada em dados reais de conversão e posição.

## 5. Desenvolvimento dos Tópicos

### A. Pedagogia: Idioma como Esporte
- **Proceduralização:** O foco é o "como" e não o "quê".
- **Frequência vs. Intensidade:** O modelo cíclico garante que termos de alta frequência sejam revisitados em diferentes contextos.
- **Ambiente Seguro:** O erro é incentivado como parte do feedback loop do treinamento motor.

### B. Marketing: A Narrativa de Inocência
- **Conceito:** O aluno não falhou por falta de talento ou esforço; ele falhou porque o método tradicional foi desenhado para ser lento.
- **Posicionamento:** A Lexis é a "fisioterapia" para quem foi "lesionado" por métodos gramaticais.
- **Copywriting:** Foco em resultados imediatos, bilinguismo executivo e autoridade técnica.

### C. Técnico: O Ecossistema Leo/Roger
- **Leo Agent (SEO):** Responsável por identificar oportunidades de "Quase Ranqueando" (Posição 5-15) e realizar upgrades de conteúdo (Upgrade Elite).
- **Roger Agent (Auditor):** Garante que o post segue o **PADRAO_BLOG_ROGER_LEO.md**. Verifica IPL (Indicador de Performance Linguística).
- **Pipeline de Publicação:** Cloudflare Worker -> GitHub Actions -> GitHub Pages. Totalmente autônomo.
- **9 Seções Obrigatórias:**
  1. Cabeçalho Bilíngue
  2. Intro com Tabela Comparativa
  3. Mapping PT/EN
  4. Nível 1 (Chunks)
  5. Nível 2 (Alta Pressão)
  6. Nível 3 (Roleplay)
  7. Erros Comuns
  8. Plano de Treino 7 Dias
  9. IPL Self-Check

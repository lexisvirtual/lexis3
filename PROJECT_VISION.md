## 1. A LINHA LÓGICA (THE CORE)

A Lexis não é uma escola de inglês; é um centro de treinamento de alta performance. A lógica do projeto segue esta cadeia ininterrupta:

1.  **Axioma Pedagógico (O "Quê"):** "Idioma não se aprende. Idioma se treina."
    - *Fundamento:* Inglês é uma habilidade motora/procedural, como dirigir ou tocar piano.
2.  **Narrativa de Marketing (O "Como"):** O conflito entre o "Método Tradicional" e o "Mecanismo 3F".
    - *Estratégia:* Inocentar o aluno (a culpa é do método acadêmico) e apresentar o treino como a única saída para a fluência real.
3.  **Motor de Escala SEO (O "Onde"):** O "Leo Protocol".
    - *Tática:* Domínio topical. Não buscamos apenas volume, buscamos "gaps" semânticos onde o Google entrega resultados fracos (fóruns/Reddit) para injetar nossa autoridade.
4.  **Infraestrutura de Elite (O "Suporte"):** Arquitetura Serverless e Estática.
    - *Eficiência:* Cloudflare Workers + GitHub Pages + Markdown. Velocidade de carregamento (LCP) como fator de ranking e conversão.

## 2. RACIOCÍNIO DETALHADO POR ETAPA

### A. O Core Pedagógico (A Base de Tudo)
O inglês é tratado como um **esporte cognitivo**.
*   **Conceito:** Ninguém aprende a nadar lendo um livro ou jogando bola assistindo palestras. A fluência reside na memória procedural (músculos e reflexos) e não na memória declarativa (regras gramaticais).
*   **O "Pulo do Gato":** Quando tratamos o idioma como habilidade física, removemos o peso emocional da "dificuldade de aprendizado". O aluno não é burro; ele apenas não está treinando os reflexos certos.

### B. Do Treino para a Venda (Mecanismo 3F)
Para materializar a pedagogia em uma oferta irresistível, usamos o **Método 3F**:
1.  **Phrase (Blocos):** O cérebro não processa palavras soltas em tempo real. Treinamos "chunks" de linguagem para reduzir o esforço cognitivo.
2.  **Fluidity (Velocidade):** O treino é cronometrado. Se você demora 5 segundos para traduzir, você não é fluente. O objetivo é o tempo de resposta zero.
3.  **Function (Realidade):** O treino simula o estresse da vida real (reuniões, viagens, pressão).
*   **Raciocínio:** Essa estrutura cria um produto tangível que o aluno consegue visualizar a evolução, diferente dos cursos "eternos".

### C. O Motor de Escala (Leo Protocol)
O SEO não é visto como "postar no blog", mas como **alocação estratégica de ativos**.
*   **Pautas de Dor:** A IA identifica termos como "travei na reunião" ou "como aprender inglês rápido" e gera conteúdos que resolvem o problema imediato enquanto vendem a imersão como a solução definitiva.
*   **Interlink:** Cada post educa o lead e o empurra para o próximo nível de consciência, criando uma máquina de pré-venda 24/7.
*   **Decisão Algorítmica:** O Leo Protocol utiliza uma fórmula de prioridade (Score) para decidir qual conteúdo deve ser criado ou reciclado, focando em "Rich Snippets" e posições de 5 a 15 no Google para saltar para o Top 3.

## 4. DESENVOLVIMENTO TÉCNICO PROFUNDO

### A. O Motor Leo Protocol (SEO Autônomo)
O Leo não é um redator; é um analista de ROI.
*   **Formula de Prioridade:** `Score = (Impressões x 0.4) + (Posição Inversa x 0.3) + (Conversões x 0.3)`.
*   **Ação em Cascata:** Se um post está na posição 12, o Leo injeta FAQs e 3 links internos para forçar a entrada no Top 10. Se o CTR está baixo, ele reescreve a Meta-Description com gatilhos de urgência.

### B. O Sistema de Imagens V9 (Hospedagem Própria)
Para garantir a integridade visual a longo prazo, o sistema eliminou links externos:
1.  **Fetch:** Busca em cascata (Pixabay -> Banco Curado).
2.  **Optimize:** Processamento via `wsrv.nl` para WebP (1200px, 80% qualidade).
3.  **Persist:** O binário é baixado e salvo em `public/img/posts/[slug].webp`.
4.  **SEO Image:** O alt text é gerado dinamicamente com as keywords principais do cluster.

### C. O Ciclo Pedagógico (Start-Run-Fly-Liberty)
A progressão é baseada em densidade cognitiva:
- **Start:** Fonética (limpar ruídos de comunicação).
- **Run:** Estruturas (automatizar gramática funcional).
- **Fly:** Vocabulário (600-700 palavras de alta frequência).
- **Liberty:** Conversação (simulações de alta pressão).
- **Imersão:** 120 horas em 14 dias (equivalente a 2 anos de escola tradicional).

# 👁️ VISÃO DO PROJETO: LEXIS ACADEMY (V9.1)

Este documento é a **Fonte Única da Verdade** (Source of Truth) para o projeto Lexis Academy. Ele consolida a estratégia pedagógica, a engenharia de marketing e a infraestrutura técnica em um único ecossistema coerente.

---

## 1. 🏗️ LINHA LÓGICA COMPLETA
A Lexis opera sob uma progressão lógica onde cada pilar sustenta o próximo:

1.  **PEDAGOGIA (A Base):** O inglês é tratado como uma **Habilidade Motora** (similar a um esporte ou instrumento), não como conteúdo acadêmico.
2.  **MARKETING (O Mecanismo):** Utiliza a **Narrativa de Inocência do Aluno**, transferindo a culpa do fracasso anterior para os métodos tradicionais e apresentando a imersão como o único caminho de alta densidade.
3.  **TÉCNICO (O Motor):** Um exército de agentes de IA (**Leo e Roger**) automatiza a criação de autoridade topical e a aquisição de leads através de SEO programático de elite.

---

## 2. 🧠 ETAPAS DO RACIOCÍNIO ESTRATÉGICO

1.  **Identificação do Gap:** O ensino tradicional foca em *conhecimento declarativo* (saber sobre a língua), enquanto a fluência exige *conhecimento procedimental* (saber usar a língua).
2.  **Mecanismo Único:** A imersão de **120 horas em 14 dias** quebra a barreira da tradução mental através da exaustão positiva e repetição deliberada.
3.  **Escalabilidade Digital:** Para atrair alunos, o sistema aloca páginas onde o Google é semanticamente fraco, usando o **Protocolo Leo** para dominar clusters de busca.
4.  **Governança de Qualidade:** O agente **Roger** audita cada peça de conteúdo, garantindo que a "Voz da Marca" e a precisão cultural brasileira sejam mantidas, evitando a "alucinação genérica" das IAs comuns.

---

## 4. 💪 ESTRATÉGIAS DE FORTALECIMENTO

*   **Indicador de Performance Linguística (IPL):** Métrica única de sucesso que une Leo e Roger. Se o post não atinge o IPL mínimo, ele entra em ciclo de reescrita automática.
*   **Ancoragem Lexical:** Técnica que obriga a IA a usar termos ensinados no início do post em cenários complexos no final, reforçando o aprendizado do leitor.
*   **Bilinguismo Estratégico:** Uso de títulos e tabelas comparativas PT/EN para facilitar a compreensão imediata enquanto introduz o "Treino Lexis".
*   **Upgrade Engine:** Posts antigos não são apenas mantidos; eles são evoluídos para o status "Elite" conforme ganham relevância.

## 5. 📚 DESENVOLVIMENTO DOS TÓPICOS

### A. Metodologia Lexis (O "Core")
- **Princípio:** Idioma se treina.
- **Método 3F:** Phrase (Contexto), Fluidity (Velocidade), Function (Aplicação Real).
- **Phases:** Start (Fonética), Run (Estruturas), Fly (Vocabulário), Liberty (Conversação).

### B. Protocolo Leo (SEO Autônomo)
- **Fórmula de Prioridade:** `Score = (Impressões x 0.4) + (Posição Inversa x 0.3) + (Conversões x 0.3)`.
- **Ações:** Otimização de Snippets, Expansão de Conteúdo Estagnado e Interlink Programático.

### C. Roger Agent (Content Guardian)
- **Função:** Auditoria de voz, tom e rigor pedagógico.
- **Veredito:** Nota 0-100. Abaixo de 60 = Reescrita forçada. Acima de 90 = Elite/Congelado.

### D. Knowledge System V9 (Infraestrutura)
- **Imagens:** Otimização via `wsrv.nl`, conversão para WebP e armazenamento local em `public/img/posts/`.
- **Hospedagem:** SPA em React com proxy via Cloudflare Workers para garantir SEO em rotas internas.

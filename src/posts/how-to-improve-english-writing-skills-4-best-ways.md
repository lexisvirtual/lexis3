**Dicas de Pronúncia**

* A pronúncia do inglês pode ser difícil para os brasileiros, pois muitas palavras têm sons diferentes dos que estamos acostumados.
* Aqui estão algumas dicas de pronúncia:
 + O som "th" é pronunciado como "t" ou "d" em inglês, dependendo do contexto.
 + O som "v" é pronunciado como "b" em inglês.
 + O som "s" é pronunciado como "z" em inglês.
 + O som "ch" é pronunciado como "tch" em inglês.
 + O som "sh" é pronunciado como "ch" em inglês.

### STRATEGIC FAQ

**Dúvidas Técnicas Respondidas**

* O que é o modo subjuntivo?
 O modo subjuntivo é usado para expressar dúvidas, incertezas ou hipóteses.
* Como usar o verbo "to be" no presente contínuo?
 O verbo "to be" no presente contínuo é usado para expressar ações que estão acontecendo no momento da fala.
* O que é o gerúndio?
 O gerúndio é um verbo que termina em "-ing" e é usado para expressar ações que estão acontecendo ao mesmo tempo que outra ação.
* Como usar o verbo "to have" no passado simples?
 O verbo "to have" no passado simples é usado para expressar ações que aconteceram no passado.
* O que é o condicional?
 O condicional é usado para expressar ações que poderiam acontecer no futuro, dependendo de certas condições.
